# Lista Encadeada

## Descrição
Implementação de uma lista encadeada simples em C utilizando alocação dinâmica de memória.

## Funcionalidades
- Inserção no início  
- Inserção no final  
- Remoção de elementos  
- Busca de elementos  
- Impressão da lista  
- Liberação de memória  

## Pré-requisitos
- Compilador C (GCC)  
- Terminal ou prompt de comando  

## Execução
```bash
gcc lista.c -o lista
./lista
